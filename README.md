# daily-news

毎朝のニュースをまとめて保管するためのリポジトリです。

## ディレクトリ構造

このリポジトリでは、ニュース記事が増えても後から振り返りやすいように、年と月ごとにディレクトリを分けて管理します。

```text
daily-news/
├── README.md              # このファイル
├── templates/             # 毎日のニュースを書く際のMarkdownテンプレート
│   └── daily_template.md
└── news/
    ├── 2026/
    │   ├── 05/
    │   │   ├── 2026-05-30.md
    │   │   └── 2026-05-31.md
    │   └── 06/
    │       ├── 2026-06-01.md
    │       └── 2026-06-10.md
    └── 2027/
```

## 運用ルール

1. **ファイル命名規則**: 必ず `YYYY-MM-DD.md` のフォーマットで作成します。（例: `2026-06-10.md`）
2. **保存場所**: `news/YYYY/MM/` ディレクトリ配下に保存します。
3. **テンプレート**: ニュースを記録する際は `templates/daily_template.md` をベースにして作成します。
